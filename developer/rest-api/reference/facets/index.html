    Facets

    api-version: 1.0

    Retrieves the list of facets.

    POST  /{collectionId}/{storageId}/_api/rest/facets

    Parameters

    Name In Required Type Description
    collectionId path true string The ID of the storage collection.
    storageId path true string The ID of the tenant.

    Responses

    Success 200

    Name Type Description
    count integer The number of returned values.
    values Facet[] The list of facets.

    Error 4xx

    Name Type Description
    message string The error message.
    type OperationResultType The result type.
    data Error[] The list of errors.

    Facet

    Name Type Description

    displayName string The display name of the view.
    views string The URI to retrieve the available views.

    error string
    warning string
